Sales Approval

Allow to configure double validation in Sale Order

AVP Technolabs

Features

This module is allow to double validation for Sale Order
Sales Configuration
  • ☑    Enable to Sales Order Approval
  • ☑    Only Manager can approve sales order

Process

Sale Configuration:
  • ☑    Enable to Sales Order Approval and set Minimum amount for Confirm order
  • ☑   Sale order can confirm only less than the configure minimum order approval amount.
  • ☑   Order go into To Approve state.
  • ☑   Only manager can approve the SO which is in the approve state..
  • ☑    Approve order by Sales Manager
  • ☑   Manage history.
